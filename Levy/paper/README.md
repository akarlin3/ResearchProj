# Levy manuscript (`paper/`)

The manuscript: an identifiability wall for the fractional order in diffusion-MRI signal decay,
and the time/space degeneracy of the joint CTRW model.

## Build

```bash
bash paper/build.sh          # consistency gate -> tectonic -> levy.pdf
```

`build.sh` runs `consistency.py` first (it FAILS the build if any number is off) and then
compiles with `tectonic` (fallback: `pdflatex` x2).

## Files

| file | role |
|---|---|
| `levy.tex` | the manuscript (house `article` style; see `% TODO-VENUE` to swap in Springer svjour3) |
| `numbers.tex` | **AUTO-GENERATED** by `consistency.py` -- every `\num*` macro; do not hand-edit |
| `consistency.py` | traceability gate: loads `results/RESULTS_CP{0,1,2}.json`, emits `numbers.tex`, verifies every manuscript number traces to a seeded result + load-bearing asserts |
| `verify_citations.py` | every `\cite` resolves to a `\bibitem`, no orphans, Ouroboros forward-cited |
| `build.sh` | gate + compile |
| `FINALIZATION_CHECKLIST.md` | the deliberate placeholders to resolve at submission |

Figures are read from `../figures/` (regenerated by `experiments/run_cp{0,1,2}.py`).

## Discipline

Every load-bearing number in `levy.tex` is a `\num*` macro defined in the auto-generated
`numbers.tex`; none is typed by hand. Regenerate the anchors with
`python experiments/run_cp{0,1,2}.py --full` and the macros with `python paper/consistency.py`.
Each CRLB result is an **identifiability** statement scoped to its regime, never an impossibility
claim.
