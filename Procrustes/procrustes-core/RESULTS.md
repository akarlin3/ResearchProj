# Procrustes — hardened results (Phases 1–3, the load-bearing numbers)

Every number below is seed-generated and regenerated by `bash reproduce.sh`, which
runs the three gate drivers (`experiments/run_phase{1,2,3}.py`) into
`results/phase{1,2,3}_*.json` and then rebuilds the manuscript (whose
`numbers.tex` is regenerated from those JSONs by `paper/consistency.py`). Ground
truth is the Lattice DRO (seed-generated, no data files). Default config: n=1000,
SNR=50, α=0.10 (90% intervals), target = the well-identified tissue diffusion **D**.
The well-identified-D\* subset is defined **exactly as Gauge's identifiable region**:
true-D\* terciles via `np.quantile([1/3, 2/3])` + `np.digitize` (bottom-2 =
identifiable, "trust D"; top tercile = Gauge's ill-posed wall). Brackets are 95%
CIs (cluster bootstrap for gaps; across-seed for AUCs and the bias ratio).

These hardened numbers supersede the CP0 scaffold figures as the load-bearing
results. The progression: CP0 (8-seed, raw-D\* cut) → Phase 1 (Gauge-exact
partition, cluster-bootstrap CIs) → Phase 3 (16-seed headline + robustness).

## GATE C — diagnostic reach (honest scope)

Worst-vs-placebo detection AUC from the bi-exp fit's residual structure, vs the
naive deployment-drift monitor (Mahalanobis on signal-shape + residual norm),
**computed on the same cohorts** (not asserted):

- **Heavy-tail:** diagnostic **0.684 [0.673, 0.694]** (structure-only 0.682;
  magnitude/χ²ᵣ 0.670) beats the naive monitor **0.550 [0.543, 0.556]**; ranks
  voxels by |D-error| (ρ = 0.169). **Detected.**
- **Pure dispersion:** diagnostic 0.578 — **below the pre-registered 0.60 floor**;
  a near-hidden channel (the naive monitor also fails). **Not detected.**
- **Tri-exp:** 0.627 — detectable but **harmless** (coverage does not break);
  detection and coverage-failure coincide only for the heavy-tail channel.

**Locked scope: a heavy-tail misspecification detector, not a universal one.**
